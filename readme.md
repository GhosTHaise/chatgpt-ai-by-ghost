This is a simple example of using chatgpt app.
>How to setup app
```shell
cd client && npm install
cd ../server && npm install
```
>How to launch app
On <b>client</b> Folder
```shell
    npm run dev
```
On <b>server</b> Folder
```shell
    npm run server
```
